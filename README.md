# 🏆 Coding Club Submission

This repository contains my complete submissions for the Coding Club challenges. The project is a full-stack, distributed real-time quiz application built with React, Node.js, Socket.io, and MongoDB.

## 📁 Repository Structure

To make evaluation as seamless as possible, the project is divided into two main directories: **Task 1** and **Task 2**. 

Inside **each** task folder, you will find exactly three things:
1. **Source Code:** The fully unzipped, ready-to-run project files.
2. **Zip Archive:** A self-contained `.zip` file of the task code.
3. **Video Demonstration:** A `.mov` video file showcasing the live functionality and real-time features of the task.

---

## 🚀 The Submissions

### [📁 Task 1: Real-Time Backend & Mobile App](./Task_1)
Focuses on the core real-time infrastructure (Node.js/Socket.io) and the adaptive mobile-first interface for participants joining the game. The mobile controller automatically transforms based on the live game state.

### [📁 Task 2: Host Portal & Presenter View](./Task_2)
Focuses on the client-side game management. This includes a robust Slide Creator workspace, a dynamic Projector screen with live bar charts, and a Host Control Dashboard to moderate the flow of the game and Q&A sessions.
